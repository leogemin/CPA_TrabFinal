## Como rodar o projeto

1. Crie e ative um ambiente virtual (opcional):

```bash
python -m venv venv
```

```bash
source venv/bin/activate
```

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

1. Rode a aplicação:

```bash
streamlit run src/main.py
```
